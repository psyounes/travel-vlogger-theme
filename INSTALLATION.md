# Guide d'Installation - Travel Vlogger Theme

## 🚀 Installation Rapide

### Étape 1 : Télécharger le thème

1. Allez sur https://github.com/psyounes/travel-vlogger-theme
2. Cliquez sur **Code** → **Download ZIP**
3. Extrayez le fichier ZIP
4. Renommez le dossier en `travel-vlogger-theme`

### Étape 2 : Téléverser sur WordPress

**Via Hostinger :**

1. Connectez-vous à votre panneau Hostinger
2. Allez dans **Fichiers** → **Gestionnaire de fichiers**
3. Naviguez vers `/public_html/wp-content/themes/`
4. **Créez** un dossier nommé `travel-vlogger-theme`
5. **Téléversez** tous les fichiers du thème dans ce dossier

**Ou via FTP :**

1. Utilisez FileZilla ou un autre client FTP
2. Connectez-vous à votre hébergement Hostinger
3. Naviguez vers `/public_html/wp-content/themes/`
4. **Glissez-déposez** le dossier `travel-vlogger-theme`

### Étape 3 : Activer le thème

1. Allez dans **Tableau de bord WordPress**
2. Cliquez sur **Apparence** → **Thèmes**
3. Vous devriez voir **Travel Vlogger**
4. Cliquez **Activer**

✅ **Le thème est maintenant actif !**

## 🔧 Configuration initiale

### Installer Elementor (OBLIGATOIRE)

1. Allez dans **Extensions** → **Ajouter**
2. Cherchez **"Elementor"**
3. Cliquez **Installer maintenant** → **Activer**

### Configurer le thème

1. Allez dans **Apparence** → **Personnaliser**
2. Configurez les sections :
   - **Couleurs** : Votre palette de couleurs
   - **Typographie** : Police et tailles
   - **Blog** : Nombre de colonnes
   - **Réseaux Sociaux** : Vos profils

### Créer les menus

1. Allez dans **Apparence** → **Menus**
2. **Créez** un nouveau menu "Menu Principal"
3. **Ajoutez** vos pages/catégories
4. **Assignez** le menu à **Menu Principal**

## 📝 Importer le contenu de démo

### Option 1 : Import rapide (Recommandé)

1. **Téléchargez** `demo-content.xml` depuis ce dépôt
2. Allez dans **Outils** → **Importer**
3. Cliquez sur **WordPress**
4. **Importez** le fichier `demo-content.xml`
5. **Assignez** les articles à votre utilisateur

✅ **6 articles de démo seront créés !**

### Option 2 : Création manuelle

#### Créer les catégories

1. Allez dans **Articles** → **Catégories**
2. Créez :
   - Europe
   - Asie
   - Afrique

#### Créer les articles

Voyez la section **"Articles de Démo"** ci-dessous.

## 📱 Widgets Elementor

### Video Hero

1. Créez une page
2. Ouvrez **Éditeur Elementor**
3. Glissez **Video Hero** dans la page
4. Entrez l'URL YouTube (ex: https://www.youtube.com/watch?v=...)
5. Personnalisez le titre et la description

### Carte Destination

1. Ajoutez le widget **Carte Destination**
2. Uploada une image
3. Entrez le nom et la localisation
4. Ajoutez un bouton CTA

### Statistiques Voyage

1. Ajoutez le widget **Statistiques Voyage**
2. Cliquez sur **Ajouter un élément**
3. Entrez vos statistiques (ex: 50 pays, 200 vidéos)

## ❓ FAQ

### Comment changer la couleur primaire ?
Allez dans **Apparence** → **Personnaliser** → **Couleurs** → **Couleur Primaire**

### Comment ajouter un logo ?
Allez dans **Apparence** → **Personnaliser** → Défiler jusqu'à **Logo du site**

### Comment créer un menu ?
Allez dans **Apparence** → **Menus** → **Créer un menu**

### Le thème ne s'affiche pas correctement ?
Vérifiez :
1. Tous les fichiers sont téléversés
2. Les permissions des dossiers sont 755
3. Videz le cache du navigateur (Ctrl+Maj+Suppr)

### Comment désactiver la barre latérale ?
Allez dans **Aparence** → **Personnaliser** → **Mise en Page** → Décochez **Afficher la barre latérale**
