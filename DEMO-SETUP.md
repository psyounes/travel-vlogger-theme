## 🚀 Étapes détaillées

### Étape 1: Télécharger et installer les images

**Pour chaque article, téléchargez une image :**

1. **Paris** → Allez sur Unsplash.com, cherchez "Paris Eiffel Tower"
2. **Bangkok** → Cherchez "Bangkok Thailand Temple"
3. **Tanzanie** → Cherchez "Safari Animals"
4. **Barcelone** → Cherchez "Barcelona Sagrada Familia"
5. **Bali** → Cherchez "Bali Temple Indonesia"
6. **Maroc** → Cherchez "Morocco Desert"

**Ajouter l'image à l'article:**
1. Éditez l'article
2. Cliquez "Définir l'image mise en avant"
3. Téléversez l'image
4. Cliquez "Définir comme image mise en avant"

---

### Étape 2: Configurer les couleurs

1. Allez dans **Apparence** → **Personnaliser**
2. Cliquez sur **Couleurs**
3. Configurez :
   - **Couleur Primaire** : `#FF6B6B`
   - **Couleur Secondaire** : `#4A90E2`
   - **Couleur du Texte** : `#333333`
   - **Fond Pied de Page** : `#1a1a1a`

---

### Étape 3: Créer le menu

1. Allez dans **Apparence** → **Menus**
2. Cliquez "Créer un menu"
3. Nommez-le "Menu Principal"
4. Ajoutez :
   - Accueil
   - Blog
   - À propos
   - Contact
5. Assignez-le à **Menu Principal**

---

### Étape 4: Créer une page d'accueil avec Elementor

1. Allez dans **Pages** → **Ajouter**
2. Titre : "Accueil"
3. Cliquez **Éditer avec Elementor**
4. Ajoutez les sections :

**Section 1: Hero**
- Widget **Video Hero**
- URL : https://www.youtube.com/watch?v=dQw4w9WgXcQ (ou votre URL)
- Titre : "Bienvenue sur Travel Vlogger"

**Section 2: Stats**
- Ajouter une **Section nouvelle**
- Ajouter le widget **Statistiques Voyage**
- Ajouter des stats :
  - 50 Pays visités
  - 200 Vidéos
  - 100K Abonnés

**Section 3: Destinations**
- Ajouter une **Section nouvelle**
- Ajouter 3 widgets **Carte Destination**
- Pour chaque :
  - Téléversez une image
  - Entrez le nom
  - Entrez la localisation
  - Entrez une description

---

### Étape 5: Configurer les réseaux sociaux

1. Allez dans **Apparence** → **Personnaliser**
2. Cliquez sur **Réseaux Sociaux**
3. Entrez vos URLs :
   - Instagram : votre profil
   - YouTube : votre chaîne
   - Facebook : votre page
   - Twitter : votre compte

## 💡 Conseils pour améliorer la démo

### Ajouter une galerie
1. Dans Elementor, ajoutez le widget **Galerie**
2. Téléversez 10-15 images de voyage
3. Organisez-les par pays

### Ajouter un formulaire de contact
1. Allez dans **Extensions** → **Ajouter**
2. Installez **WPForms Lite** (gratuit)
3. Créez un formulaire de contact simple
4. Créez une page "Contact" avec le formulaire

### Ajouter un slider (carrousel)
1. Dans Elementor, utilisez le widget **Carousel** natif
2. Ajoutez vos meilleures photos
3. Configurez l'autoplay

## ❓ Dépannage

### Le thème n'apparaît pas
- Vérifiez que tous les fichiers sont dans `/wp-content/themes/travel-vlogger-theme/`
- Videz le cache du navigateur
- Réactivez le thème

### Elementor ne s'ouvre pas
- Vérifiez que Elementor est installé et activé
- Mettez à jour WordPress
- Essayez de désactiver les autres extensions

### Les images n'apparaissent pas
- Vérifiez les permissions des dossiers (755)
- Vérifiez que les images sont au bon format (JPG, PNG)
- Essayez de compresser les images
