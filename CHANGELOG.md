## 📦 Installation

### Étape 1 : Téléverser le thème

1. Téléchargez le dossier `travel-vlogger-theme`
2. Allez sur Hostinger → Fichiers → Gestionnaire de fichiers
3. Naviguez vers `/public_html/wp-content/themes/`
4. Créez un dossier `travel-vlogger-theme`
5. Téléversez tous les fichiers du thème

### Étape 2 : Activer

1. Allez dans **Apparence** → **Thèmes**
2. Trouvez **Travel Vlogger**
3. Cliquez **Activer**

### Étape 3 : Installer Elementor (Optionnel)

1. Allez dans **Extensions** → **Ajouter**
2. Cherchez **"Elementor"**
3. Installez et activez

⚠️ **S'il y a un crash :**
- Désactivez tous les autres plugins
- Mettez à jour WordPress
- Réactivez Elementor

## 🎛️ Utiliser le Panneau Travel Vlogger

### Menu Principal

Dans le tableau de bord WordPress, vous verrez **"Travel Vlogger"** dans le menu gauche.

Cliquez dessus pour accéder :

### 1. **Paramètres Généraux**

- Logo du site
- Tagline
- Articles par page
- Colonnes de grille (1-4)
- Afficher/masquer la barre latérale
- Afficher/masquer les extraits

### 2. **Couleurs**

- Couleur primaire
- Couleur secondaire
- Couleur du texte
- Couleur du pied de page

### 3. **Démo**

**Importer la démo en 1 clic :**
1. Allez dans **Travel Vlogger** → **Démo**
2. Cliquez **🚀 Importer la Démo**
3. Confirmez

✅ **6 articles + 3 catégories seront créés !**

## 🎨 Personnaliser le thème

### Ajouter un logo

1. Allez dans **Travel Vlogger** → **Paramètres Généraux**
2. Entrez l'URL de votre logo
3. Sauvegardez

### Changer les couleurs

1. Allez dans **Travel Vlogger** → **Couleurs**
2. Cliquez sur les carrés de couleur
3. Choisissez vos couleurs
4. Sauvegardez

### Configurer le blog

1. Allez dans **Travel Vlogger** → **Paramètres Généraux**
2. Ajustez :
   - Articles par page
   - Colonnes de grille
   - Affichage de la barre latérale
   - Affichage des extraits

## 🐛 Dépannage

### Le site ne s'affiche pas

- Vérifiez que tous les fichiers sont téléversés
- Videz le cache (Ctrl+Maj+Suppr)
- Réactivez le thème

### Elementor crash

- Mettez à jour WordPress
- Mettez à jour Elementor
- Désactivez les autres plugins
- Réactivez progressivement

### Les couleurs ne changent pas

- Videz le cache du navigateur
- Videz le cache du plugin de cache (si présent)
- Sauvegardez à nouveau
