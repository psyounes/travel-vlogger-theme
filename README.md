# Travel Vlogger Theme

Un thème WordPress moderne et professionnel pour les vlogeurs de voyage.

## 🎯 Fonctionnalités

✅ **Customizer WordPress avancé**
- Personnalisation complète des couleurs
- Contrôle de la typographie
- Configuration du layout
- Paramètres blog
- Intégration réseaux sociaux

✅ **Compatible Elementor**
- 3 widgets personnalisés
- Catégorie thème dédiée
- Design responsive

✅ **Responsive Design**
- Mobile-first
- Tested sur tous les appareils
- Menu mobile intuitif

✅ **SEO Friendly**
- Structure HTML sémantique
- Métadonnées optimisées
- Permaliens personnalisables

## 📋 Widgets Elementor

### 1. Video Hero
- Affiche une vidéo YouTube en fond
- Contenu textuel superposé
- Entièrement personnalisable

### 2. Carte Destination
- Image de destination
- Titre et localisation
- Description et bouton CTA
- Effets au survol

### 3. Statistiques Voyage
- Affiche les statistiques
- Format en colonnes
- Répéteur pour ajouter plusieurs stats

## 🛠️ Installation

1. Téléchargez le thème
2. Allez dans Apparence → Thèmes
3. Cliquez sur "Ajouter"
4. Téléversez le fichier zip
5. Activez le thème

## ⚙️ Configuration

Après activation :

1. **Apparence → Personnaliser** pour configurer les couleurs
2. **Apparence → Menus** pour créer les menus
3. **Extensions** pour installer Elementor (gratuit)
